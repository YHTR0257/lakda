# Knowledge Retrieval Assistant System (知識検索アシスタントシステム)

## Directory Overview (ディレクトリ概要)

このディレクトリは、特定のドメインに関する質問に対して、関連情報を検索し、コンテキストを付与した回答を生成する汎用的なRAGシステム「Knowledge Retrieval Assistant System」の開発プロジェクトです。

現在は設計フェーズであり、プロジェクトの計画、アーキテクチャ設計、機能要件などがドキュメントにまとめられています。

## Key Files (主要ファイル)

- `docs/project_design.md`: プロジェクトの全体像、目的、タイムライン、機能要件、アーキテクチャ、リポジトリ構成などが定義された最も重要なドキュメントです。
- `pyproject.toml`: Pythonプロジェクトとしての基本的な設定が記述されています。現時点では依存関係は定義されていません。
- `docs/architecture.md`, `docs/features.md`, `docs/api_endpoint.md`: これらのファイルは将来のドキュメントのために用意されていますが、現在は空です。
